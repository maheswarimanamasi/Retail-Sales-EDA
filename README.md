# Retail Sales Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to identify customer purchasing behavior, product performance, and sales trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Information
- Rows: 1000
- Columns: 9
- Missing Values: 0
- Duplicate Records: 0

## Analysis Performed
- Data Cleaning
- Statistical Analysis
- Customer Demographics Analysis
- Product Category Analysis
- Monthly Sales Trend Analysis
- Correlation Analysis
- Top Customer Identification

## Key Insights
1. Clothing and Electronics generate the highest revenue.
2. Sales vary across months indicating seasonal patterns.
3. Quantity purchased positively impacts total revenue.
4. Male and female customer participation is balanced.
5. A small group of customers contributes significantly to revenue.

## Recommendations
- Increase stock for high-performing categories.
- Run promotions during low-sales months.
- Develop customer loyalty programs.
- Use demographic-based marketing strategies.
